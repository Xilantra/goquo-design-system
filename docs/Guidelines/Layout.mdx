## Responsive Breakpoint
| Target            | Name    | Pixel     | Rem     |
| :---------------: | :-----: | :-------: | :-----: |
| Phones            | xs      | 0         | 0       |
| Landscape Phones  | sm      | 576px     | 36rem   |
| Tablets           | md      | 768px     | 48rem   |
| Desktops          | lg      | 1024px    | 64rem   |
| Large Desktops    | xl      | 1200px    | 75rem   |

<small>*xl is available in CSS only.</small>


## Flexbox Grid
Taken from [react-styled-flexboxgrid](https://github.com/loicmahieu/react-styled-flexboxgrid).
Responsive modifiers enable specifying different column sizes, offsets, alignment and distribution at xs, sm, md & lg viewport widths.

`xs={12} sm={3} md={2} lg={1}`
